# 🎮 人生RPG - Life System

> 不是记录人生，而是运行人生的角色扮演游戏

把你的真实生活变成一款RPG游戏！通过记录能量、睡眠、学习、运动来让你的角色升级，解锁成就，提升四维属性。

## ✨ RPG核心系统

### 🧬 角色系统

每个玩家都有一个独特的角色，拥有：

- **等级系统** - 从Lv.1新手开始，通过获得经验值不断升级
- **经验值(EXP)** - 完成日常任务获得经验，升级后提升属性
- **称号** - 根据等级自动获得称号（新手🌱 → 探索者🔍 → 生命大师🏆）
- **年龄** - 记录你的游戏天数，见证成长轨迹

### 📊 四维属性

| 属性 | 图标 | 说明 | 提升方式 |
|------|------|------|----------|
| **力量 (STR)** | 💪 | 体能、肌肉力量 | 记录运动活动 |
| **智力 (INT)** | 🧠 | 脑力、学习能力 | 记录学习活动 |
| **体力 (VIT)** | 🏃 | 能量容量、耐力 | 运动+休息 |
| **精神 (SPI)** | ✨ | 睡眠质量、心理状态 | 优质睡眠 |

所有属性范围：0-100

### ⚡ 衍生属性

- **HP（生命值）** = 100 + 力量×2 + 体力×3
- **最大能量** = 100 + 体力×5
- **战斗力** = 力量 + 智力 + 体力 + 精神

### 🎯 经验值获取

| 活动 | 经验值 | 额外奖励 |
|------|--------|----------|
| 记录能量 | +5 EXP | - |
| 运动 | +15 EXP | 力量+0.5, 体力+0.3 |
| 学习 | +15 EXP | 智力+0.5 |
| 优质睡眠（≥8分） | +20 EXP | 精神+0.5, HP+10 |
| 普通睡眠 | +10 EXP | 精神+0.5, HP+10 |
| 连续天数奖励 | +50 EXP | - |
| 完成每日目标 | +30 EXP | - |
| 解锁成就 | +100 EXP | - |

### 🏆 成就系统

#### 入门成就
- **初次记录** 🎯 - 完成第一次能量记录
- **七日修行** 🔥 - 连续记录7天
- **十级冒险者** ⬆️ - 达到10级

#### 进阶成就
- **月度坚持** 💪 - 连续记录30天
- **二十级英雄** 🦸 - 达到20级
- **睡眠大师** 😴 - 连续7天睡眠质量≥8分
- **能量管理者** ⚡ - 连续7天能量水平≥7分

#### 大师成就
- **全属性50+** 💎 - 所有属性达到50以上
- **力量巨人** 💪 - 力量属性达到80
- **智慧贤者** 🧠 - 智力属性达到80

## 🎮 游戏玩法

### 每日流程

```
早上起床
  ↓
📊 查看角色面板（等级、属性、HP、能量）
  ↓
😴 记录昨晚睡眠 → 获得经验值 → 精神属性提升
  ↓
⚡ 记录早晨能量水平 → 获得经验值
  ↓
（白天活动...）
  ↓
⚡ 记录下午能量水平
  ↓
⚡ 记录晚上能量水平 → 如果有运动/学习 → 对应属性提升
  ↓
💡 查看系统建议（失眠预警、能量分析）
  ↓
🏆 检查新解锁成就
  ↓
睡眠
```

### 升级机制

- 每次升级随机提升一项属性 +2
- 等级越高，所需经验值指数增长
- 升级时自动更新称号
- HP和能量上限自动重新计算

### 称号晋升路线

```
Lv.1-4    新手 🌱
Lv.5-9    学徒 📚
Lv.10-14  探索者 🔍
Lv.15-19  修行者 🧘
Lv.20-29  进化者 🚀
Lv.30-39  优化专家 ⭐
Lv.40-49  自律宗师 👑
Lv.50+    生命大师 🏆
```

## 📱 界面展示

### 主界面
- **顶部Header** - 显示游戏名称、当前等级、称号、连续天数、总经验
- **角色卡片** - 大尺寸角色展示区，包含头像动画、等级徽章、经验条、HP条、能量条、四维属性
- **快速统计** - 4个RPG风格卡片：连续修炼、战斗力、睡眠修复、能量储备
- **系统提示** - AI生成的任务建议和警告
- **任务板块** - 能量追踪和睡眠记录
- **冒险指南** - 游戏机制说明

### 角色详情面板
- **基础信息卡** - 等级、旅程天数、平均属性、成就数量
- **属性分析** - 最强属性和需提升属性
- **成就墙** - 已解锁和未解锁成就展示，彩色/灰度区分

## 🚀 快速开始

### 安装和运行

```bash
# 进入项目目录
cd life-system

# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

访问 [http://localhost:3000](http://localhost:3000) 开始你的人生RPG之旅！

### 新手教程

1. **创建角色** - 首次打开自动创建Lv.1角色
2. **记录睡眠** - 点击"睡眠记录"卡片，记录昨晚睡眠情况
3. **记录能量** - 在"能量追踪"中记录早/中/晚能量水平
4. **选择活动** - 如果有运动或学习，勾选对应活动类型
5. **观察成长** - 查看角色卡片，经验值增加、属性提升、可能升级！
6. **解锁成就** - 持续坚持，解锁各种成就奖励

## 🎨 设计理念

### 游戏化元素
- **即时反馈** - 每次记录立即获得经验值和属性提升

- **能量管理** - 教会你识别和管理自己的能量状态
- **持续优化** - 通过数据分析找到最佳生活模式

## 💡 使用技巧

### 快速升级策略
1. **每天三次能量记录** - 稳定获得经验
2. **选择运动/学习活动** - 双倍经验+属性提升
3. **保持优质睡眠** - 单次最高经验值
4. **维持连续天数** - 解锁成就奖励

### 属性平衡发展
- 不要只专注一项属性
- 四维属性都提升到50+可解锁成就
- 力量和智力通过主动活动提升
- 体力和精神通过睡眠和休息提升

### 成就收集
- 优先完成"七日修行"和"十级冒险者"
- 睡眠质量相关成就需要持续优化作息
- 高等级成就需要长期投入
