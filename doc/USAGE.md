# 🎮 人生RPG - 使用指南

## 快速开始

```bash
cd /sessions/awesome-dreamy-ride/mnt/life-system
npm run dev
```

访问：http://localhost:3000

## 🎯 核心玩法

### 1. 角色成长系统

你的角色从 **Lv.1 新手🌱** 开始，通过完成日常任务获得经验值升级。

**当前版本实现的功能：**
- ✅ 实时经验值系统
- ✅ 自动升级机制
- ✅ 四维属性（力量、智力、体力、精神）
- ✅ HP和能量系统
- ✅ 称号系统（10个等级称号）
- ✅ 成就系统（10个成就）
- ✅ 角色年龄追踪

### 2. 经验值获取方式

| 操作 | 经验值 | 属性提升 |
|------|--------|----------|
| 记录能量（每次） | +5 | - |
| 标注运动活动 | +15 | 力量+0.5, 体力+0.3 |
| 标注学习活动 | +15 | 智力+0.5 |
| 优质睡眠（8-10分） | +20 | 精神+0.5, HP+10 |
| 普通睡眠（5-7分） | +10 | 精神+0.5, HP+10 |
| 解锁成就 | +100 | - |

### 3. 属性系统

#### 四维属性（0-100）

- **力量💪** - 通过运动提升
- **智力🧠** - 通过学习提升
- **体力🏃** - 通过运动+休息提升
- **精神✨** - 通过优质睡眠提升

#### 衍生属性

- **HP（生命值）** = 100 + 力量×2 + 体力×3
- **最大能量** = 100 + 体力×5
- **战斗力** = 四项属性总和

## 📝 每日使用流程

### 早上（推荐7-9点）

1. **打开应用**
   - 查看角色卡片，观察当前等级和属性
   - 查看"系统提示"，了解昨日建议

2. **记录睡眠**
   - 点击右侧"😴 睡眠记录"卡片
   - 输入昨晚入睡和起床时间
   - 对睡眠质量打分（1-10）
   - 如果失眠，选择失眠原因
   - 点击"保存睡眠记录"
   - **立即获得经验值！**（10-20 EXP）

3. **记录早晨能量**
   - 点击左侧"⚡ 能量追踪"卡片
   - 确认时段为"早晨"
   - 拖动滑块选择能量水平（1-10）
   - 点击"保存记录"
   - **获得 5 EXP**

### 下午（推荐14-16点）

1. **记录下午能量**
   - 点击"能量追踪"
   - 切换到"下午"时段
   - 选择能量水平
   - 如果有运动或学习，勾选对应活动
   - **如果选择运动/学习，额外获得 15 EXP 和属性提升！**

### 晚上（推荐20-22点）

1. **记录晚间能量**
   - 切换到"晚上"时段
   - 记录当前能量水平
   - 选择活动类型

2. **查看数据分析**
   - 查看"近7天能量趋势"图表
   - 观察连续天数（Streak）
   - 查看角色属性变化

3. **阅读系统建议**
   - 如果晚上能量>7，系统会建议运动
   - 根据建议调整作息

## 💡 高效升级技巧

### 快速升级策略（前10级）

1. **每天完成三次能量记录**
   - 早晨 → 5 EXP
   - 下午 → 5 EXP
   - 晚上 → 5 EXP
   - **每天基础 15 EXP**

2. **选择活动类型**
   - 下午或晚上记录时
   - 勾选"运动🏃"或"脑力🧠"
   - **每个活动额外 15 EXP**
   - 如果两个都做：15 + 15 + 15 = **45 EXP/天**

3. **保持优质睡眠**
   - 睡眠质量8分以上
   - **20 EXP/天**

4. **理想情况下每天获得**
   - 基础记录：15 EXP
   - 活动记录：30 EXP（运动+学习）
   - 优质睡眠：20 EXP
   - **总计：65 EXP/天**

5. **升级时间表**
   - Lv.1 → Lv.2：100 EXP（2天）
   - Lv.2 → Lv.3：150 EXP（2-3天）
   - Lv.3 → Lv.4：225 EXP（3-4天）
   - Lv.4 → Lv.5：337 EXP（5天）
   - ...
   - **Lv.10大约需要7-10天**

### 属性提升策略

1. **力量提升**
   - 每次记录能量时选择"运动🏃"
   - 每次 +0.5 力量
   - 从10提升到50需要：80次运动记录
   - 如果每天1次：80天
   - 如果每天2次（下午+晚上）：40天

2. **智力提升**
   - 记录能量时选择"脑力🧠"或"工作💼"
   - 每次 +0.5 智力
   - 策略同力量

3. **体力提升**
   - 运动会顺带提升体力（+0.3）
   - 多运动，体力自然上升

4. **精神提升**
   - 每次优质睡眠 +0.5 精神
   - 保持良好作息

5. **平衡发展建议**
   - 不要只做运动或学习
   - 每天至少一次运动+一次学习
   - 保证睡眠质量
   - 四维属性均衡提升

## 🔍 常见问题

### Q1: 如何快速升级？
A: 每天坚持三次能量记录+选择活动类型+优质睡眠，可获得60+ EXP/天

### Q2: 属性提升太慢怎么办？
A: 每次记录时务必选择活动类型（运动/学习），这是属性提升的关键

### Q3: 升级后有什么奖励？
A: 每次升级随机提升一项属性+2，称号自动更新，HP和能量上限提高

### Q4: 成就解锁后有什么用？
A: 每个成就奖励100 EXP，可以加速升级

### Q5: 数据会丢失吗？
A: 所有数据存储在浏览器localStorage中，除非清空浏览器数据，否则不会丢失

### Q6: 能创建多个角色吗？
A: 当前版本只支持单角色，未来会支持多角色系统

### Q7: 睡眠质量怎么评分？
A:
- 8-10分：睡得很好，精力充沛
- 5-7分：普通睡眠
- 1-4分：睡眠质量差，建议记录失眠原因

### Q8: 连续天数断了怎么办？
A: 连续天数会重新从0开始计算，坚持每天记录是关键
