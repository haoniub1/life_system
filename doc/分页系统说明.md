# 📑 多标签页系统说明

## 🎯 系统架构

应用现在分为4个主要标签页，每个页面都有独立的功能区域：

```
🎮 人生RPG
├── 🎮 总览 (Overview)
├── 📋 任务系统 (Tasks)
├── 🎁 奖励商店 (Rewards)
└── 📜 历史记录 (History)
```

---

## 📄 各标签页详解

### 1. 🎮 总览页面

**核心功能：** 查看角色状态、记录日常数据

**包含模块：**
- ✨ **角色卡片** - 显示等级、经验、四大属性
- 📊 **详细属性面板** - 可折叠，查看成就和属性分析
- ⚠️ **用进废退警告** - 提示哪些属性即将衰退
- ⚡😴 **能量/助眠切换** - 双轨制能量系统
- 📈 **快速统计** - 连续天数、战斗力、平均睡眠等
- 📊 **能量趋势图** - 查看历史能量变化
- ⚡ **能量追踪** - 记录早中晚能量水平
- 😴 **睡眠记录** - 记录睡眠时间和质量
- 📖 **新手指南** - 使用说明

**适用场景：**
- 每天打开应用的第一站
- 快速了解当前状态
- 记录日常数据

---

### 2. 📋 任务系统页面

**核心功能：** 创建、管理、完成任务

**包含功能：**
- ➕ **创建任务**
  - 设置标题、描述
  - 选择分类（运动/学习/社交/工作/休息/其他）
  - 选择类型（自定义/每日/每周）
  - 配置多个奖励（经验/金币/属性）

- 📝 **任务列表**
  - 进行中的任务（可完成、可删除）
  - 已完成的任务（灰色显示）

- 💰 **金币显示**
  - 实时查看当前金币余额

**示例任务：**
```
任务：阅读书100页
分类：🧠 学习
奖励：
  - 智力 +1
  - 金币 +10
  - 经验值 +15
```

**适用场景：**
- 制定每日/每周计划
- 完成任务获得奖励
- 查看待办事项

---

### 3. 🎁 奖励商店页面

**核心功能：** 用金币购买道具，提升角色

**商品分类：**

#### 💪 属性提升（50金币/个）
- 💪 力量药水 - 永久+1力量
- 🧠 智力药水 - 永久+1智力
- ❤️ 体力药水 - 永久+1体力
- ✨ 精神药水 - 永久+1精神

#### ⚡ 能量恢复
- 🧪 脑力恢复剂（30金币）- 恢复50脑力
- ⚗️ 体力恢复剂（30金币）- 恢复50体力
- 💊 全能恢复剂（80金币）- 完全恢复脑力和体力

#### ✨ 特殊道具
- ⭐ 经验加速器（100金币）- 立即获得100经验值
- 😴 助眠增强剂（60金币）- 增加50点双助眠值
- 🍀 幸运金币（120金币）- 随机属性+1~3

**功能特点：**
- 💰 金币余额显示
- 🔍 分类筛选
- 💳 一键购买
- 📊 购买次数统计

**适用场景：**
- 金币充足时提升属性
- 紧急恢复能量值
- 快速获得经验升级

---

### 4. 📜 历史记录页面

**核心功能：** 查看过往成就和数据

**统计数据：**
- 📋 总完成任务数
- 💰 总获得金币
- ⭐ 总获得经验值
- 🎮 当前等级

**时间线展示：**
```
📅 今天
  ✅ 14:30 完成"阅读书100页"
     💰 +10金币  🧠 +1智力  ⭐ +15经验值

  ✅ 10:15 完成"跑步5公里"
     💰 +15金币  💪 +1力量  ❤️ +1体力

📆 昨天
  ✅ 20:00 完成"写日记"
     💰 +5金币  ✨ +1精神

📚 更早
  ✅ 2026-01-19 完成"学习编程"
     💰 +20金币  🧠 +2智力
```

**成就进度：**
- ✅ 完成第1个任务
- ⬜ 完成10个任务
- ⬜ 完成50个任务
- ✅ 今日完成3个任务
- ✅ 累积100金币
- ⬜ 累积1000金币

**适用场景：**
- 回顾自己的成长轨迹
- 查看任务完成情况
- 激励持续行动

---

## 🎨 设计特点

### 1. 顶部固定导航栏
- 始终可见，方便切换
- 高亮当前标签页
- 响应式设计（移动端显示图标）

### 2. 统一视觉风格
- 紫色/蓝色渐变主题
- 卡片式布局
- 圆角阴影
- 悬停动画

### 3. 独立页面状态
- 每个标签页独立管理内容
- 数据实时同步
- 无需重新加载

## 💡 使用流程示例

### 场景1：早晨起床后

```
1. 打开应用 → 🎮 总览
   - 查看角色状态
   - 查看用进废退警告

2. 切换到 📋 任务系统
   - 创建今日任务：
     * 阅读30分钟
     * 跑步5公里
     * 工作2小时

3. 开始执行任务...
```

---

### 场景2：完成任务后

```
1. 切换到 📋 任务系统
   - 点击"✓ 完成"按钮
   - 获得奖励提示

2. 切换到 📜 历史记录
   - 查看刚才完成的任务
   - 查看总金币增加

3. 如果金币充足 → 切换到 🎁 奖励商店
   - 购买属性药水
   - 提升角色战斗力
```

---

### 场景3：晚上睡前

```
1. 🎮 总览 → 记录睡眠
   - 输入入睡和醒来时间
   - 评分睡眠质量

2. 📜 历史记录
   - 查看今日完成的所有任务
   - 查看获得的总奖励

3. 🎮 总览 → 能量/助眠切换
   - 切换到"😴 助眠视角"
   - 查看助眠值是否充足
```

## 📱 响应式设计

- **桌面端（>768px）**
  - 4个标签页横向排列
  - 完整显示标签文字

- **移动端（<768px）**
  - 标签页只显示图标
  - 内容区域全宽显示

## 🎯 总结

多标签页系统让应用更加清晰、有条理：

- **🎮 总览** = 仪表盘，快速掌握状态
- **📋 任务系统** = 行动中心，规划和执行
- **🎁 奖励商店** = 成长加速，用金币提升
- **📜 历史记录** = 回顾过往，激励前行

**每个页面都有明确的目的，用户可以根据当前需求快速切换！** 🎮✨
