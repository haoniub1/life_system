# 🚀 快速启动指南

## 方式一：一键启动

```bash
cd /sessions/awesome-dreamy-ride/mnt/life-system && npm run dev
```

然后访问: http://localhost:3000

## 方式二：分步启动

```bash
# 1. 进入项目目录
cd /sessions/awesome-dreamy-ride/mnt/life-system

# 2. 确认依赖已安装（首次使用）
npm install

# 3. 启动开发服务器
npm run dev
```

## 🎮 首次使用

1. **打开浏览器** - 访问 http://localhost:3000
2. **查看角色** - 你将看到一个 Lv.1 的新手角色
3. **记录睡眠** - 点击右侧"睡眠记录"开始
4. **记录能量** - 点击左侧"能量追踪"
5. **观察成长** - 每次记录后角色会获得经验值！

## 📊 数据说明

- 所有数据保存在浏览器 localStorage
- 刷新页面数据不会丢失
- 清空浏览器数据会重置角色

## 🏆 快速升级技巧

1. 每天记录能量3次（早/中/晚）→ 15 EXP
2. 选择活动类型（运动/学习）→ 额外30 EXP
3. 记录优质睡眠 → 20 EXP
4. **每天可获得 60+ EXP**
