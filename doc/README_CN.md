# 🧬 人生系统

> 不是记录人生，而是运行人生

一个基于能量管理、睡眠追踪和认知优化的个人生活管理系统。

## ✨ 核心功能

### 1. 能量管理中心 ⚡
- 📊 每日三次能量记录（早/中/晚）
- 🏃 活动标注（运动/脑力/社交/工作/休息）
- 📈 7天能量趋势曲线图
- 💡 智能能量分析和建议

### 2. 睡眠质量追踪 😴
- ⏰ 记录入睡和起床时间
- ⭐ 睡眠质量评分（1-10）
- 🔍 失眠原因归因分析
  - 能量过剩
  - 压力焦虑
  - 身体不适
  - 其他原因
- 📊 睡眠时长和质量趋势

### 3. 智能建议系统 💡
- 🤖 基于能量和睡眠数据生成每日建议
- ⚠️ 失眠风险预警
- 📊 能量消耗模式分析
- 🎯 个性化优化建议

### 4. 统计面板 📊
- 🔥 连续充实天数（Streak）
- ⭐ 总积分累计
- 😴 平均睡眠质量
- ⚡ 平均能量水平

## 🚀 快速开始

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

应用将在 [http://localhost:3000](http://localhost:3000) 启动

### 构建生产版本

```bash
npm run build
npm start
```

## 🛠️ 技术栈

- **前端框架**: Next.js 15 + React 19
- **语言**: TypeScript
- **样式**: Tailwind CSS
- **状态管理**: Zustand + 本地持久化
- **图表**: ECharts
- **日期处理**: date-fns

## 📖 使用指南

### 第一步：记录能量

1. 在首页找到"能量追踪"卡片
2. 选择时段（早晨/下午/晚上）
3. 拖动滑块选择能量水平（1-10）
4. 选择主要活动类型
5. 点击"保存记录"

### 第二步：记录睡眠

1. 每天早上记录昨晚的睡眠
2. 输入入睡时间和起床时间
3. 评分睡眠质量（1-10）
4. 如果失眠，标注失眠原因
5. 点击"保存睡眠记录"

### 第三步：查看分析

- 📊 **能量曲线图**：查看7天能量趋势
- 💡 **今日建议**：系统自动生成的优化建议
- 📈 **统计卡片**：长期数据统计和趋势

## 🎯 设计理念

基于以下8个核心理解：

1. **能量管理与睡眠** - 能量消耗不足导致失眠
2. **大脑工作内存管理** - 及时清空工作内存
3. **长期记忆优化** - 经历压缩与数据结构优化
4. **超量恢复** - 用进废退的双刃剑
5. **力量层次** - 表层力量vs里层力量
6. **运动+信念** - 主动健康管理
7. **欲望驱动与信念系统** - 内在动力机制
8. **幸福归因重构** - 保持内在控制感
