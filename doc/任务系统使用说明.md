# 📋 任务系统使用说明

## ✨ 功能概述

任务系统让你可以创建自定义任务，完成任务后获得各种奖励，提升你的角色属性和金币！

---

## 🎯 如何创建任务

### 示例：你的需求 "阅读书100页 增加知识1 奖励金币10"

1. **点击"创建任务"按钮**
2. **填写任务信息：**
   - **任务标题**：`阅读书100页`
   - **任务描述**（可选）：`完成《xxx》的第X章`
   - **任务分类**：选择 `🧠 学习`
   - **任务类型**：
     - 自定义：一次性任务
     - 每日任务：每天可以重复
     - 每周任务：每周可以重复

3. **设置奖励：**
   - 奖励1：`🧠 智力 +1`
   - 奖励2：`💰 金币 +10`
   - 点击"+ 添加奖励"可以添加更多奖励

4. **点击"创建任务"完成**

---

## 🎁 奖励类型

| 图标 | 类型 | 说明 |
|------|------|------|
| ⭐ | 经验值 | 增加角色经验，帮助升级 |
| 💰 | 金币 | 游戏货币（未来可用于购买道具）|
| 💪 | 力量 | 提升体能属性 |
| 🧠 | 智力 | 提升脑力属性 |
| ❤️ | 体力 | 提升能量容量 |
| ✨ | 精神 | 提升精神状态 |

## 🔄 任务类型说明

### 自定义任务
- 一次性任务
- 完成后会移到"已完成"列表
- 适合：特定的目标、项目、事件

### 每日任务
- 每天自动重置
- 完成后第二天又可以继续完成
- 适合：每天要做的习惯（如阅读、运动）

### 每周任务
- 每周自动重置
- 适合：每周的固定活动

---

## 💡 使用技巧

1. **合理设置奖励**
   - 简单任务：经验值10-15，金币5-10
   - 中等任务：经验值20-30，金币15-25
   - 困难任务：经验值40-50，金币30-50

2. **配合"用进废退"机制**
   - 创建学习类每日任务 → 保持智力不衰退
   - 创建运动类每日任务 → 保持力量和体力不衰退
   - 优质睡眠后创建任务 → 保持精神不衰退

3. **任务分解**
   - 大任务拆分成小任务
   - 例如："完成论文" → "文献阅读"、"大纲编写"、"初稿撰写"等

## 🎮 与RPG系统的联动

- **完成任务 → 获得经验值 → 升级 → 解锁称号**
- **累积金币 → 未来可以购买道具、皮肤等**
- **属性提升 → 增强角色战斗力**
- **连续完成 → 解锁成就**

## 📊 查看任务统计

任务系统会自动记录：

- ⭐ 通过任务获得的经验值
- 📈 每日/每周完成率

## 🚀 开始使用

1. 打开应用，滚动到"任务系统"区块
2. 点击"+ 创建任务"
3. 按照你的需求填写任务信息
4. 完成任务后点击"✓ 完成"按钮
5. 查看你获得的奖励！
